# 博物馆AI特效合作方案展示

## 文件说明
- `index.html` - 微信兼容版H5页面（推荐，纯CSS无外部依赖）
- `mobile_index.html` - 备用版本

## 部署到GitHub Pages

### 方法一：创建新仓库
1. 在GitHub创建新仓库 `museum-ai-effects`
2. 上传所有文件
3. Settings → Pages → Source: main branch → Save
4. 等待1-2分钟，访问 `https://你的用户名.github.io/museum-ai-effects/`

### 方法二：使用现有仓库
直接上传到任意仓库的gh-pages分支即可

## 微信访问
部署完成后，将链接发送到微信即可直接打开
